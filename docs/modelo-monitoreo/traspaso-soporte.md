# Traspaso a soporte del modelo de monitoreo

## Objetivo

Documentar lo que soporte necesita saber para operar y diagnosticar usando el nuevo modelo de monitoreo.

## Información requerida

| Elemento | Descripción |
| --- | --- |
| Producto | Nombre, ambiente soportado y responsable. |
| Fuente | Origen, frecuencia y criticidad. |
| Dashboard | Link, paneles principales y criterios de interpretación. |
| Funciones | Wrappers o helpers usados por el dashboard. |
| Runbook | Pasos de diagnóstico y evidencia mínima. |
| Escalamiento | Equipo, canal y criterios para derivar. |

## Validación

El traspaso se considera suficiente cuando soporte puede reproducir el diagnóstico inicial sin depender de conocimiento tribal.
