# Catálogo de funciones

Este catálogo fue construido a partir de los archivos KQL presentes en `refactor_ada_optimized/law_functions` y `refactor_ada_optimized/grafana_wrappers`. Los propósitos se infieren desde nombre, carpeta y uso observado; cuando el repositorio no permite confirmarlo completamente, se indica **Por validar**.

> Nota: no se reproducen rutas productivas sensibles, IDs de suscripción ni credenciales. Para detalles exactos de workspace, revisar los archivos KQL con permisos adecuados.

## Categorías

- Helpers cross-producto: lógica reutilizable entre productos.
- Helpers específicos: lógica auxiliar acoplada a un producto o dominio.
- Fuentes: funciones de acceso a LAW/tablas.
- Dominios: funciones que calculan estado por dominio.
- Wrappers Grafana: queries livianas para paneles o variables.

- ADA, NOTPII y SIROSAG siguen en general la separación `sources/helpers/domains/wrappers`.

- ADA AMG tiene funciones de dominio en carpeta `domain` en singular y wrappers asociados; el validador local lo marca como brecha de convención.

- `fn_prd_mlp_ada_amg_jobs_status_detail` está referenciada por wrappers/funciones ADA AMG, pero no se encontró definición en `law_functions`; queda **Por validar**.
